# huffman_tree
a sample code for compressing a text file by **huffman_tree** algorithm

***

also this code can extract the compressed file!

***

for run gui:
```
./startGUI.sh
```

### run in terminal

###### for compress a file:
```
./start.sh 0 file_name
```
###### for extract a compressed file:
```
./start.sh 1 file_name
```
